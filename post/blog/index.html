My first Blog about use hugo

May 14, 2020 · 1 min read

用Hugo来创建自己的博客，命令如下图所示：hugo new site myblog 注：myblog为文件夹名字 ![usehugo]](/img/hugo/usehugo.png)

下载设置主题，去https://themes.gohugo.io/选择自己喜欢的主题进行下载

启动博客命令，输入 hugo server -t m10c –buildDrafts

然后是创建文章，在根目录下创建，如图

写博客，推荐用Vscode，操作.md文件（markdown文件）

然后再根目录下输入指令 hugo server -t m10c –buildDrafts

再打开http://localhost:1313，效果就出现了

hugo图片路径： 在Markdown图片路径中，以static为根目录写全路径 示例：假设测试baseurl为http://localhost:1313/，图片位置：/static/img/hugo/123.png，那么在Markdown中写作![图片说明]图片路径(/img/hugo/123.png)。这时，在浏览器就可以看到图片了
